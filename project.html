Restaurant Website
The Taste of Qin Yun

Project Scope & Requirements

1. Target Users

  • Target users are restaurant customers looking to learn more about the restaurant, including location, menu, and general information.
  • Users will access the app on both mobile and desktop devices.
  • Users should be able to order food online through the website using an external food ordering service.
  • The app should be easy to use, quick to navigate, and mobile-friendly.

2. Static Information

  • Company / Restaurant name
  • Address and location
  • Contact information (phone number, email)
  • Menu items and descriptions
  • Information about the restaurant

3. Design Elements

  • Simple and clean layout
  • Easy navigation with clear sections
  • Strong focus on displaying the menu and food items
  • Information should be easy to find
  • Avoid cluttered layouts or confusing navigation

4. Data Management

  • Menu information
  • Restaurant hours / schedule

5. Interactive Elements

  • Interactive map showing restaurant location
  • Online ordering functionality
